# 管理员功能

:::tip 一点说明
推荐使用用[系统管理面板](./system.md)创建管理员账号，
:::

管理员账户会新增 2 个菜单（应用管理，网站监控）。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/9d5a24a56459586ec05be474bbbec5b5)

## 应用管理

### 概况

展示访问数据，操作日志和一些系统信息。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/df18d2bca095de62093d89e627147587)

### 用户

查看一些用户相关的基本信息，包含（账号状态，密码重置，消息推送等）。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/9699e648861ed28a8f58f0f9e13c0386)

### 需求

展示一些用户反馈的需求信息和反馈的问题。

### 配置

#### 路由禁用

入口：应用管理 > 配置 > 禁用路由管理

![](https://cdn.upyun.sugarat.top/mdImg/sugar/fe60f7a8a438e52bb7c56241c30bfd95)

禁用对应页面将无法访问，会有如下提示。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/c81cd3f31b536bc75fd0d459a08663a7)

禁用注册后，会同时隐藏页面上注册相关入口，和禁用注册接口的调用。

#### 细节配置

配置一些系统上的功能细节。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/2bd1358568f40c7b84a3c4e1a3b6adf2)

![](https://cdn.upyun.sugarat.top/mdImg/sugar/f5509885d7e790fbcff21cf8deef9238)

## 网站监控

监控网站的基本情况 和 稳定性。

![](https://cdn.upyun.sugarat.top/mdImg/sugar/056ae989946c67358a43db004db723b8)

![](https://cdn.upyun.sugarat.top/mdImg/sugar/d64d0a6acaadc41d43c1f3684077a323)
